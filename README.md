# FarahTechnology Solutions

Site institucional e de vendas da **FarahTechnology Solutions** — empresa de desenvolvimento web, aplicações e gestão de redes sociais.

🌐 **[farah-tecnology.vercel.app](https://farah-tecnology.vercel.app)**

---

## Stack

- **React 18** — componentes e lógica da UI
- **Vue 3** — accordion FAQ (Composition API)
- **Vite 5** — build e dev server
- **CSS puro** — design system próprio, sem framework

## Estrutura

```
src/
├── components/   # 15 componentes React
├── vue/          # FaqAccordion.vue (Vue 3)
├── hooks/        # useInView, useCountUp
├── styles/       # global.css, components.css
└── utils/        # helpers.js
```

## Rodar localmente

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # build de produção
```

## Deploy

Hospedado na **Vercel**. Todo `git push` na branch `main` dispara deploy automático.
